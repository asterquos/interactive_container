集装箱装载管理系统 v1.0

==== 快速开始 ====

【首次使用】
1. 确保电脑已安装 Python 3.8 或更高版本
   - 如果没有，请从 https://www.python.org 下载安装
   - 安装时务必勾选 "Add Python to PATH"

2. 双击运行 "setup_windows.bat" 
   - 自动安装运行环境
   - 首次运行需要几分钟下载组件

3. 双击运行 "run_windows.bat" 启动程序

【日常使用】
直接双击 "run_windows.bat" 启动程序

【制作exe文件】
双击 "build_windows.bat" 生成独立的exe程序
生成的文件在 dist 文件夹中，可以拷贝给其他人使用

==== 软件功能 ====

【数据导入】
文件菜单 → 导入Excel文件
支持标准的箱子清单Excel表格

【集装箱管理】
集装箱菜单 → 新建集装箱
可以创建多个集装箱进行对比

【装载操作】
- 从左侧列表拖拽箱子到中央集装箱
- 拖拽箱子调整位置
- 右键点击箱子可以旋转或移回左侧
- 实时显示重量平衡信息

【保存和加载】
文件菜单 → 保存项目 / 加载项目
可以保存装载方案供以后使用

【示例数据】
测试菜单 → 加载示例数据
快速体验软件功能

==== 系统要求 ====

- Windows 7/8/10/11
- Python 3.8+ (首次安装会自动配置)
- 内存 4GB 以上
- 硬盘空间 500MB

==== 常见问题 ====

问题：程序无法启动
解决：
1. 检查是否安装了Python
2. 重新运行 setup_windows.bat
3. 以管理员身份运行

问题：杀毒软件报毒
解决：这是误报，请添加到白名单

问题：中文显示乱码
解决：检查系统区域设置是否为中文

问题：Excel文件无法导入
解决：
1. 确保Excel文件格式正确
2. 文件不要被其他程序占用
3. 使用示例数据测试功能

==== 技术支持 ====
